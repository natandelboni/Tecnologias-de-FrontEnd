# Fábrica de Projetos Valdir

Este projeto é uma landing page desenvolvida para apresentar o grupo e o projeto da disciplina de Fábrica de Projetos. A página contém informações sobre a disciplina, a empresa, o tema do projeto e os membros do grupo.

## Tecnologias Utilizadas

- **HTML**: Estrutura básica da página.
- **Tailwind CSS**: Framework CSS para estilização e layout responsivo.
- **JavaScript**: (opcional) Para futuras interações dinâmicas (não utilizado nesta versão).
